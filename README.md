## Activity management social media web app built with ASP.NetCore and React

## How to install from Terminal:

<sub>
mkdir NewFolder
cd NewFolder
git clone https://github.com/NominNomi/Reactivities.git
cd Reactivities
cd API
dotnet run
cd ..
cd client-app
npm install vite
npm start
browser: http://localhost:3000
<sub/>
