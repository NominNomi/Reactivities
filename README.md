## How to install from Terminal:

mkdir NewFolder  
cd NewFolder  
git clone https://github.com/NominNomi/Reactivities.git  
cd Reactivities  
cd API  
dotnet run  
cd ..  
cd client-app  
npm install vite  
npm start  
browser: http://localhost:3000
